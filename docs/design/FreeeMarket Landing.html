// order lifecycle

From cart to delivered, the contract holds the line.

Marketplace.sol · unaudited
01 // PAY
Funded

Buyer approves USDC and calls buy(). The amount is escrowed; the encrypted shipping address travels off-chain.

02 // SHIP
In transit

Seller decrypts the address, ships, and sends a tracking code back over the same encrypted channel.

03 // CONFIRM
Completed

Buyer calls confirmReceipt(). 100 % of escrow releases to the seller — or auto-releases after 14 days.

04 // BRANCH
Disputed

Either party can open a dispute. The arbiter refunds the buyer or pays the seller — never themselves.
